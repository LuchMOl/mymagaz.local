Так в принципе во всех языках. Потому данные хранят в других местах.
Например хранят в файлах.
Но в основном юзают специальные хранилища которые называют базы данных.
Есть разные типы (можно почитать погуглив "типы баз данных", но я не советую это делать), мы будем юзать только реляционную бд (база данных). В частности ее реализацию mysql. Mysql это просто название фирмы которая сделала бд. Есть другие типа PostgreSQL, MicrosoftSQL, OracleDB итд.
Мы юзаем mysql потому что он халявный и вполне могучий. Для мелких проектов вполне подходит (мелкий это даже розетка может быть например).
Для общения с базами данных используется специальный язык SQL. То есть он как бы унифицирован. И по сути зная как и какие надо выполнять комманды на sql, ты можешь спокойно работать с любой бд производителей что я написал выше. То есть ты изи можешь перейти в проект на постгре или оракл.
Визуально хранение данных в реляционнной бд можно представить в виде таблиц, как в экселе. Например есть таблица пользователей. Там хранятся данные по всем пользователям сайта, или таблица товаров.
Обычно мы используем 4 основных запроса к бд: создать, прочитать, обновить, удалить. (CRUD create read update delete)
